# yuitest-bot

現在の https://twitter.com/yuitest_bot の実装です。

## 方針

GitHub Workflows の schedule 書式を利用して、一定時間ごとに投稿を行うようにします。
そうすれば GitHub Actions が生きている限りは bot も稼働し続けるはずです。
